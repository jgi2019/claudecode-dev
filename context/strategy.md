# AIオンボーディング戦略・Base44

## AIオンボーディング事業
- 開始：2025年3〜4月
- コアコンセプト：「組織のAIネイティブ化支援」
- ターゲット：SMB（中小企業）
- 最大の目的：JGIの利益効率を上げ、半年〜1年で大きく加速させる

### 背景・課題認識
- 近い将来、AIエージェント＝AI従業員が当たり前になる
- Claude CoworkやClaude Codeがその先駆け
- SMBオーナーにとってAIは生存戦略に不可欠
- しかし国内の多くの企業はAI活用で足踏みしている
- ここを助けることがJGIの役割

### 提供内容
- AIの基礎教育と具体的活用事例の提供
- Claude Code / Claude Coworkを使ったエージェント構築支援
- Base44を活用した簡単・迅速な実装支援
- 組織レジリエンスの設計（誰が欠けても1日でリカバリできる体制）

### 組織レジリエンスの考え方
- SMBは「辞めてもらっては困る人」で形成されている
- その人が急に離脱しても止まらない組織を作る
- AIをOSに置き、クラウドと連携し、ハンドオフが残る状態
- 1日あればリカバリできる＝組織の継続的活動を守る

## Base44
- イスラエル発のバイブコーディングサービス（Wixグループ）
- JGIは国内法人で初めて公式パートナーに登録
- 江頭淳は世界44名のアンバサダーに日本国内で初めて就任
- AIオンボーディング事業との組み合わせが核心
- 非エンジニアでも簡単にアプリ・ツールを作れる基盤として活用

## 今後の方向性
- Claude Code / Cowork + Base44の組み合わせを軸にしたサービス設計
- 自社でのインフラ実践→証明→横展開
- 日本のBase44アンバサダー第1号としてのポジション確立
- DiffyなどのAIツールも組み合わせた複合的な提案
